# Code

To denote a word or phrase as code, enclose it in tick marks (&#96;).

```markdown
At the command prompt, type `nano`.
```

At the command prompt, type `nano`.

## Code Blocks (Fenced)

Use three backticks (&#96;&#96;&#96;) on the lines before and after the code to
depict the code as a code block.

```markdown
{
  "firstName": "John",
  "lastName": "Smith",
  "a
}
```
